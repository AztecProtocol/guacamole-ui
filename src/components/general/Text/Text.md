#### Basic Usage

```jsx static
<Text
  text="Hello"
  size="m"
/>
```

```js
<Text
  text="Hello"
  size="m"
/>
```


#### Sizes

```js

  text="Text size xxs"
  size="xxs"
/>

<Text
  text="Text size xs"
  size="xs"
/>

  text="Text size s"
  size="s"

<br />
<Text
  text="Text size xl"
  size="xl"

<br />

  size="xxl"
/>
```

#### Responsive Sizes

```js
<Text
  text="size={{ m: 's', s: 'xl' xs: 'm' }}"
  size={{ m: 's', s: 'xl', xs: 'm' }}
/>
```
